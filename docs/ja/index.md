# Weight Goal Bot の使い方

[@my_weight_goal_bot](https://t.me/my_weight_goal_bot) は、1人でも複数のメンバーでも、チャット内でお互いに見守りながら体重目標を続けられるようにします。写真付きの記録は人同士で進捗を確認する材料になり、ボットは体重を記録して週間の推移を整理し、記録忘れやペースの乱れを防ぐリマインダーを送ります。週ごとの節目を達成すると、楽しいカピバラ実績が解除されます。

## はじめ方

1. ボットをグループに追加します。
2. 現在の写真に `@my_weight_goal_bot 93.05 kg` と添えて送ります。
3. 目標体重を返信します。例：`80 kg`
4. 期限を返信します。例：`2026年12月31日`
5. 内容を確認して「作成する」を押します。

日付、目標体重、各チェックポイントで減らすグラム数をまとめた週ごとの計画が届きます。

## 毎週の記録

新しい写真のキャプションでボットをメンションし、現在の体重を書いて送ります。例：`@my_weight_goal_bot 88.3 kg`。記録後、今週の目標判定と進捗グラフが届きます。いつでも記録できますが、週1回以上がおすすめです。

## コマンド

| コマンド | 内容 |
|---|---|
| `/goal` | 新しい目標を作成、または現在の目標を変更します。以前の目標は履歴に残ります。 |
| `/status` | 目標、今週のチェックポイント、進捗グラフを表示します。 |
| `/schedule` | 修正済みの週間計画をもう一度表示します。 |
| `/settings` | 言語を選択します。 |
| `/help` | 短いコマンド一覧を表示します。 |

グループでは、ボットをメンションしてコマンドを送るか、`/command@my_weight_goal_bot` を使用してください。

進行中の目標がある場合、`/goal` は最初にボタンで変更の確認を求めます。確認後、ボットをメンションするか案内メッセージに返信して、新しい開始時点の写真と体重を送ってください。以前の目標が履歴に移るのは、新しい目標の作成が完了してからです。

## 計画・グラフ・実績

- 通常の1週間は50 g単位でできるだけ均等に配分され、最初と最後の端数週は少なくなる場合があります。
- 最終目標体重と期限は常に正確です。
- 画像の生成は1ユーザーにつき1分に1回です。
- 木曜日に今週の目標をお知らせし、週内の記録がまだなければ日曜日の10:00にも再度リマインドします。
- 週目標を達成するたびに、カピバラの実績が順番に解除されます。
- 言語は `/settings`、またはボットとの個別チャットにあるボタンからいつでも変更できます。

このボットは継続を応援するためのもので、医療上の助言ではありません。
